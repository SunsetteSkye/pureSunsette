# Icon palette assignments (hand-authored)

Per-species party/OW icon palettes, icon shading convention (NOT battle-sprite shading):
- idx0 = white (transparent for OBJ)
- idx1 = light/highlight
- idx2 = accent
- idx3 = black

Format: `species (variant): idx1 hex (RGB triple) / idx2 hex (RGB triple)`
(idx0 = RGB 31,31,31, idx3 = RGB 0,0,0 unless noted)

## Shorthands

- "cream" = F8F8F0 = RGB 31,31,30 (recurring highlight)
- DEFAULT: a mon assignment covers BOTH non-alt and alt unless the user explicitly scopes it ("default only", "non-alt", or gives the alt a separate color). Do not ask.
